# Proyect 2: 

Con Arduino: Cuando el detector de presencia no detecte ninguna persona, que haya un led rojo encendido. Cuando dicho detector compruebe que hay una persona que se encienda una luz led verde y muestre por pantalla la hora en la que ha detectado la presencia. Ayuda: https://playground.arduino.cc/Code/PIRsense
Con la placa Waspmote y el sensor de luminosidad: Cuando el sensor de luminosidad detecte que es de noche (habrá que establecer un nivel de luminosidad que sea de noche), que encienda el led verde. Mientras tanto, que esté apagado. 

Materiales necesarios:
Placa Waspmote
Placa de prototipado
miniUSB Cable
Luz led rojo
Luz led verde
1x Waspmote Events Board
1x PIR Sensor
1x Luminosity Sensor
WIFI Pro Onchip
Placa Arduino.
Otro sensor PIR.
Batería
